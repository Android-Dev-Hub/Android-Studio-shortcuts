# Android Studio Shortcuts

A list of commonly used keyboard shortcuts in Android Studio to speed up development.

---

## General

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Open Project | Ctrl + Shift + O | Cmd + Shift + O |
| Save All | Ctrl + S | Cmd + S |
| Settings / Preferences | Ctrl + Alt + S | Cmd + , |
| Recent Files | Ctrl + E | Cmd + E |
| Navigate to Class | Ctrl + N | Cmd + O |
| Navigate to File | Ctrl + Shift + N | Cmd + Shift + O |
| Navigate to Symbol | Ctrl + Alt + Shift + N | Cmd + Alt + O |
| Search Everywhere | Double Shift | Double Shift |

---

## Editing

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Reformat Code | Ctrl + Alt + L | Cmd + Option + L |
| Optimize Imports | Ctrl + Alt + O | Ctrl + Option + O |
| Comment Line | Ctrl + / | Cmd + / |
| Comment Block | Ctrl + Shift + / | Cmd + Option + / |
| Duplicate Line | Ctrl + D | Cmd + D |
| Delete Line | Ctrl + Y | Cmd + Delete |
| Move Line Up/Down | Alt + Shift + Up/Down | Option + Shift + Up/Down |

---

## Navigation

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Go to Declaration | Ctrl + B | Cmd + B |
| Go to Implementation | Ctrl + Alt + B | Cmd + Option + B |
| Go to Super Method | Ctrl + U | Cmd + U |
| Find Usages | Alt + F7 | Option + F7 |
| Next Error | F2 | F2 |
| Previous Error | Shift + F2 | Shift + F2 |

---

## Running

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Run App | Shift + F10 | Ctrl + R |
| Debug App | Shift + F9 | Ctrl + D |
| Stop App | Ctrl + F2 | Cmd + F2 |
| Build Project | Ctrl + F9 | Cmd + F9 |
| Rebuild Project | Ctrl + Shift + F9 | Cmd + Shift + F9 |

---

## Refactoring

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Rename | Shift + F6 | Shift + F6 |
| Change Signature | Ctrl + F6 | Cmd + F6 |
| Extract Method | Ctrl + Alt + M | Cmd + Option + M |
| Extract Variable | Ctrl + Alt + V | Cmd + Option + V |
| Extract Constant | Ctrl + Alt + C | Cmd + Option + C |
